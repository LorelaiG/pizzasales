Built a complete end-to-end data analyst portfolio project using SQL Server and Power BI. The project focused on analyzing pizza sales data by importing and cleaning raw CSV data, writing SQL queries to generate KPIs and business insights, and building an interactive Power BI dashboard. Key metrics included total revenue, average order value, total pizzas sold, total orders, and average pizzas per order. Advanced SQL concepts such as aggregate functions, GROUP BY, subqueries, filtering, and sorting were used to answer business requirements. In Power BI, the data was connected directly to SQL Server, transformed using Power Query, and visualized through dynamic charts, KPIs, slicers, and interactive filters. The dashboard included daily and monthly sales trends, sales by pizza category and size, and top/bottom performing pizzas by revenue, quantity, and orders. The project emphasized data validation between SQL and Power BI to ensure reporting accuracy and demonstrated practical business intelligence, data cleaning, visualization, and dashboard design skills
